## Unholy:
+ Придерживается идеальной ротации
+ Авто баф
+ Авто смена стоек в зависимости от ситуации
  + вне комбата включает анхоли стойку, чтобы максимально профитно бурстануть, после бурста включает блад
+ Авто таргет + автотаргет пета
+ Логика стаков на синдре
+ Авто ловля дебафов с помощью зеленки
  + Пламя Легиона(Джара)
  + Роящиеся тени(Лана)
+ Авто обновление мора
+ Авто смена соло/аое ротации
+ Гаргу и усиление рунического оружия использует исключительно в босса, либо в любой таргет при включении CD
+ Поддерживает бафф отчаяния и баф от т4
+ Профиль поддерживает 4т4 2т4 и ротацию без кусков

## Balance:
+ Придерживается идеальной ротации
+ Дает звездопад только в босса, либо в любой таргет при включении CD
+ Автотаргет
+ Авто волшебный огонь
+ Авто камень и дубовая кожа в критические моменты с функцией ттд(тайм ту дай)
+ Умная дубовая кожа - прожимается под определенные ситуации боссов
  + Ледяная метка(Синдрагоса)
  + Жатва душ(Лич)
  + Едкая гниль(Тухлопуз)
  + Адский удар(Логова Грулла хм)
  + Пронизывающий холод(Ануб)
+ В т4 рой нажимается только на перебежках
+ Профиль поддерживает 4т4 2т4 и ротацию без кусков

## Restoration:
+ Хилит придерживаясь идеальной ротации
+ Умный буйный рост
+ Покровительство Природы и Восстановление в наилучшее время
+ Автопрохил меток
+ Заранее обхотывает в нужные моменты(например восстановление в того, кто будет заходить в портал на Копыте(кара хм)
+ Раскидывает по миликам омоложения
+ При включении AOE начинает спамить омоложение, если не придумает ничего другого
+ При включении CD начинает отдавать жизнецвет под прок
+ Поддерживает обхот Мутировавшего поганища в цлк(Для включения функции поганище должно быть в фокусе)
+ Режим умного исцеления на 3 фазе ануба
+ Режим исцеления Валитрии
+ Авто смерч на леди + идеальное продление в 0.1(начинает каст пока игрок еще в смерче)
+ Авто декурс определенных дебафов
+ Авто декурс всего что видит(опционально)
+ Авто стоп каст под кики боссов
+ Логика стаков на синдре
+ Умное использование Дубовой кожи под определенные ситуации боссов
+ Природная стремительность + Целительное прикосновение с логикой ttd
+ Автоматическое использование трынь и инженерских перчаток в нужные моменты
+ Умное использование дубовой кожи, покровительства природы и камня здоровьяе
+ Функция хилить всех, для аутдоров и тд.(хилит даже тех кто не в рейде)
