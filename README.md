## Restoration:
+ Хилит придерживаясь идеальной ротации
+ Умный буйный рост
+ Покровительство Природы и Восстановление, для кражи хпса
+ Автопрохил меток
  + Испепеление плоти(Джара)
  + Метка смерти(Мороуз)
  + Метка падшего воителя(Саурфанк)
  + Жатва душ(Лич)
+ Заранее обхотывает в нужные моменты
  + Ледяная метка(Синдрагоса)
  + Пронизывающая стужа(Синдрагоса)
  + Метка демона(Копыто хм)
  + Гравитационная бомба(Физрук)
+ Раскидывает по миликам омоложения
+ При включении AOE начинает спамить омоложение, если не придумает ничего другого
+ При включении CD начинает отдавать жизнецвет под прок
+ Поддерживает обхот Мутировавшего поганища в цлк
  + Для включения функции поганище должно быть в фокусе
  + Так же в простои кидает в поганище восстановление, для увеличения общего хпса
+ Режим умного исцеления на 3 фазе ануба
  + При включении, приоритетом прохиливает метки и танка
+ Режим исцеления Валитрии
  + При включении хилит только валитрию
  + В выключеном режиме прохиливает валитрию в свободное время
+ Авто смерч + идеальное продление в 0.1
  + Начинает заного кастовать смерч до его спадения
  + Если Природная стремительность не в кд, то даст инста смерч
+ Авто декурс
+ Умное использование Дубовой кожи под определенные ситуации боссов
  + Ледяная метка(Синдрагоса)
  + Жатва душ(Лич)
  + Едкая гниль(Тухлопуз)
  + Адский удар(Логова Грулла хм)
  + Пронизывающий холод(Ануб)
+ Природная стремительность + Целительное прикосновение с логикой ttd
+ Автоматическое использование трынь и инженерских перчаток в нужные моменты
+ Умное использование дубовой кожи, покровительства природы и камня здоровьяе
+ Функция хилить всех, для аутдоров и тд...
  + Хилит даже тех, кто не в рейде
